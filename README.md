# The Breath of the Beloved

A contemplative Rumi & Shams Tabrizi quote generator — single-page, self-contained, zero dependencies.

## Design System

Informed by the [UI/UX Pro Max](https://uupm.cc) design intelligence toolkit.

| Principle | Application |

| **Dark Mode (OLED)** | `#0A0A0F` background, gold `#C9A84C` accent, minimal glow |

## Features

- **1,212 authentic quotes** from Rumi's *Masnavi*, *Divan-e Shams*, *Rubaiyat*, *Fi-hi Ma Fihi*, and Shams Tabrizi's *Maqalat*
- **Filter by author** — toggle between All / Rumi / Shams Tabrizi
- **Light / Dark mode** — manual toggle, persists in localStorage, respects `prefers-color-scheme`
- **Copy to clipboard** — one-click copy with tooltip feedback
- **Share** — native share sheet (Web Share API) with clipboard fallback
- **Keyboard shortcut** — press Space for a new quote
- **Daily email signup** — localStorage-persisted subscription form
- **Mouse-tracking glow** — subtle radial gradient follows cursor on card hover
- **Responsive** — 375px / 768px / 1024px breakpoints
- **44px touch targets** on all interactive elements

## Quick Start

Open `rumi-shams.html` in any browser. No build step, no server needed.

## Deploy

Any static host works. Cloudflare Pages recommended:

1. Push repo → Cloudflare Dashboard → Workers & Pages → Create → Pages
2. Connect repo, framework preset = **None**, output dir = `.`
3. Rename `rumi-shams.html` to `index.html` for root-URL serving

## Customize

- **Colors**: Edit `:root` / `.light-mode` CSS custom properties
- **Quotes**: Modify the `quotes` array in the `<script>` block (or re-run `generate_quotes.py`)
- **Fonts**: Swap Google Fonts URLs in the `<link>` tag
- **Animations**: Adjust `--transition` variable (currently `250ms`)
