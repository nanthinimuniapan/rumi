# The Breath of the Beloved

A contemplative quote generator featuring Rumi, Shams Tabrizi, Kahlil Gibran & Rabindranath Tagore — single-page, self-contained, zero dependencies.

## Design System

| Principle | Application |
|---|---|
| **Minimalism** | Clean layout, high contrast, essential elements only |
| **Dark Mode (OLED)** | `#0A0A0F` background, gold `#C9A84C` accent, subtle glow |
| **Aurora UI** | 3 animated gradient blobs (18-26s cycle) for depth |
| **Solid card** | `#12121C` background, subtle `box-shadow`, radial mouse-tracking glow |
| **Editorial Classic** | Spectral for poetry, Figtree for UI text |
| **Micro-interactions** | 250ms transitions, button lift on hover, rotate refresh icon |
| **Accessible & Ethical** | `prefers-reduced-motion`, `focus-visible` rings, keyboard nav, `aria-*` attributes |
| **Spacing** | 8dp rhythm (`4/8/16/24/32/48/64px`) |

## Features

- **2,038 authentic quotes** from Rumi, Shams Tabrizi, Kahlil Gibran, and Rabindranath Tagore
- **Filter by author** — toggle between All / Rumi / Shams Tabrizi / Gibran / Tagore
- **Light / Dark mode** — manual toggle, persists in localStorage, respects `prefers-color-scheme`
- **Copy to clipboard** — one-click copy with tooltip feedback
- **Share** — native share sheet (Web Share API) with clipboard fallback
- **Keyboard shortcut** — press Space for a new verse
- **Mouse-tracking glow** — subtle radial gradient follows cursor on card hover
- **Responsive** — 600px / 480px / 380px + landscape breakpoints, 100dvh
- **44px touch targets** on all interactive elements
- **prefers-reduced-motion** — disables all animations for vestibular disorders

## Quick Start

Open `rumi-shams.html` in any browser. No build step, no server needed.

## Deploy

Any static host works. Cloudflare Pages recommended:

1. Push repo → Cloudflare Dashboard → Workers & Pages → Create → Pages
2. Connect repo, framework preset = **None**, output dir = `.`
3. Rename `rumi-shams.html` to `index.html` for root-URL serving

## File Structure


## Customize

- **Colors**: Edit `:root` / `.light-mode` CSS custom properties
- **Quotes**: Modify the `quotes` array in the `<script>` block (or re-run `generate_quotes.py`)
- **Fonts**: Swap Google Fonts URLs in the `<link>` tag
- **Animations**: Adjust `--transition` variable (currently `250ms`)
